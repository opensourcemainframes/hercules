Hercules Version 2: System Messages: LG - Log functions

This page describes messages related to the log funtions of the Hercules S/370, ESA/390, and z/Architecture emulator.

Messages

HHCLG001S Error redirecting stdout: error
Meaning
The stdout stream could not be redirected to the system logger, the error is described by error.
Issued by
logger.c, function InitStreams
HHHCLG002E Error reading syslog pipe: error
Meaning
An error occurred while reading the syslog pipe. The error is described by error.
Issued by
logger.c, function logger_thread
HHCLG003E Error writing to hardcopy file: error
Meaning
The error indicated by error occurred while writing to the hardcopy logfile.
Issued by
logger.c, function logger_thread
HHCLG004E Error duplicating stderr: error
Meaning
Stdout could not be redirected to stderr. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG005E Error duplicating stdout: error
Meaning
Stderr could not be redirected to stdout. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG006E Duplicate error redirecting hardcopy log: error
Meaning
The error described by error occurred whilst redirecting the hardcopy log.
Issued by
logger.c, function logger_init
HHCLG007S Hardcopy log fdopen failed: error
Meaning
An attempt to open a stream for the hardcopy log failed. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG008S logbuffer malloc failed: error
Meaning
An instorage buffer for the system log could not be obtained. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG009S Syslog message pipe creation failed: error
Meaning
An attempt to create the pipe for the system logger failed. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG010S Syslog write message pipe open failed: error
Meaning
An attempt to open a stream for the logger pipe failed. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG011S Syslog read message pipe open failed: error
Meaning
An attempt to open a stream for the logger pipe failed. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG012S Could not create logger_thread: %s
Meaning
An attempt to open a stream for the logger pipe failed. The error is described by error.
Issued by
logger.c, function logger_init
HHCLG013I logger_thread started: tid=threadid, pid=processid
Meaning
The system logger thread has been started. Its thread id is threadid, and its process id is processid.
Issued by
logger.c, function logger_thread
HHCLG014I Hardcopy file forced closed.
Meaning
The hardcopy logfile has been closed as a result of too many consecutive i/o errors. The preceding HHCLG003E message describes the error(s).
Issued by
logger.c, function logger_thread
HHCLG015I logger_thread ended
Meaning
Informational
Issued by
logger.c, function logger_thread
HHCLG016S Error creating command input stream: error
Meaning
The error described by error occurred while duplicating or opening the command input stream.
Issued by
logger.c, function InitStreams
HHCLG017S Error creating logfile hardcopy stream: error
Meaning
The error described by error occurred while duplicating or opening the logfile hardcopy stream.
Issued by
logger.c, function InitStreams
HHCLG018S Error creating physical screen stream: error
Meaning
The error described by error occurred while duplicating or opening the physical screen stream.
Issued by
logger.c, function InitStreams
HHCLG019S Error creating daemon-mode state stream: error
Meaning
The error described by error occurred while duplicating or opening the daemon-mode state stream.
Issued by
logger.c, function InitStreams
