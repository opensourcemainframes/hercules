The Hercules System/370, ESA/390, and z/Architecture Emulator

Hercules is an open source software implementation of the mainframe System/370 and ESA/390 architectures, in addition to the new 64-bit z/Architecture. Hercules runs under Linux, Windows 98, Windows NT, Windows 2000, Windows XP, and Mac OS X 10.3 and later.

Hercules was created by Roger Bowler and is maintained by Jay Maynard. Jan Jaeger designed and implemented many of the advanced features of Hercules, including dynamic reconfiguration, integrated console, interpretive execution and z/Architecture support. A dedicated crew of programmers is constantly at work implementing new features and fixing bugs.

The CVS version

The complete source code for the current development version of Hercules is also available via anonymous access from our CVS source code repository. The CVS root is
CVSROOT=:pserver:anonymous@cvs.hercules-390.org:/usr/cvs/hercules.
Doing a checkout on module "hercules" will get you the source for all of Hercules.

No password is required; just press enter when asked:

cvs login
(Logging in to anonymous@hercules-390.org...)
CVS password:
cvs -z9 co hercules
cvs server: Updating hercules

Please note that this will get you the current development version of Hercules, which is not release quality and thus might not even work (since it's still under development). If you want the current, stable, release version of Hercules (i.e. one that is known to work properly), then use the previously mentioned links instead.

Read the file README.CVS for additional and updated instructions for building the development version.
